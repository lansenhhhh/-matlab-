# 微环腔热效应仿真

该项目旨在模拟微环腔在热效应作用下的光脉冲传播。通过引入腔内功率和热动态方程，研究光脉冲在腔内的演变过程。代码使用MATLAB编写，包含参数设置、热效应计算和脉冲光谱的处理。

## 主要功能
- **参数设置**: 配置微环腔的基本参数，例如损耗、非线性系数、耦合系数等。
- **热动态计算**: 通过欧拉方法计算微腔内的温度变化，考虑热膨胀和热光效应。
- **光脉冲模拟**: 模拟泵浦光在腔内的传播，计算腔内功率变化和光谱。

## 运行要求
- MATLAB环境
- 确保`matlab.mat`文件在运行目录中

## 使用说明
1. 打开MATLAB。
2. 确保所有依赖文件和数据已准备好。
3. 运行主代码文件，即可开始仿真。

## 参数说明
- `Po`: 输入功率（瓦特）
- `kappa`: 腔内耦合系数
- `alpha`: 一圈损耗
- `gamma`: 非线性系数
- `beta2`: 二阶色散系数
- `Q`: 微环品质因数
- `lambda_pump`: 泵浦光波长

## 结果展示
- 仿真结果包括腔内功率随时间变化的图像，展示热效应对光脉冲的影响。
